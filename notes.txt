- Reactjs - lean and focused component based UI library. Certain features for example routing are added via community packages
- Angular - Complete component based UI framework, packed with features. Uses typescript. Can be overkill for smaller projects.
- Vuejs - complete component based ui framework, includes core features. A bit less popular than react and angular.
